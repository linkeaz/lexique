# Guide éditorial

## But
Rendre les termes techniques compréhensibles sans perdre en exactitude.

## Style
- Phrase courte.
- Vocabulaire accessible.
- Précision technique maintenue.
- Éviter le jargon non expliqué.

## Bonne entrée
- Définition courte immédiatement utile.
- Définition longue orientée usage concret.
- Alias pertinents seulement.
- Liens "related" réellement utiles.
